# api
API to get info from Weather service (openweathermap.org) and list of events, births and deaths from Wikipedia (https://en.wikipedia.org/wiki/January_1)

Weather forecast can be taken here
https://myawesomeapp.me/weather/docs
Example of request
https://myawesomeapp.me/weather/forecast?lat=44.34&log=10.99&date=2022-10-20


List of Events, Births and Deaths by day from Wikipedia can be taken here
https://myawesomeapp.me/wiki/docs
Example of request
https://myawesomeapp.me/wiki/getEventsByDay?date=January_1
